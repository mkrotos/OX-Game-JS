# OX Game
## Browser version of OX game written in JavaScript <br><br>
### How to run
1. Install dependencies using command: <br>
    `npm i`
2. Start http server: <br> 
    `npm start`
3. Open browser on adress: <br>
   `http://127.0.0.1:8080`


### How to run tests
1. Install dependencies using command: <br>
    `npm i`
2. Run command: <br>
   `npm test`
